# Windows - Hardening Básico - Part 72 - Microsoft Store

Garanta que 'Disable all apps from Microsoft Store' esteja atribuido como 'Disabled'

Essa configuração define como iniciar todos os aplicativos da Microsoft Store que vieram pré-instalados ou foram baixados.

O estado recomendado para essa configuração é: Desativado.

Observação: essa configuração de política se aplica somente ao Windows 10 ou edições Enterprise e Education posteriores.

Computer Configuration\Policies\Administrative Templates\Windows Components\Store\Disable all apps from Microsoft Store

Garanta que 'Turn off Automatic Download and Install of updates' esteja atribuido como 'Disabled'

Essa configuração ativa ou desativa o download e a instalação automáticos de atualizações de aplicativos da Microsoft Store.

O estado recomendado para essa configuração é: Desativado.

Manter seu sistema devidamente atualizado pode ajudar a protegê-lo contra vulnerabilidades de dia zero.

Computer Configuration\Policies\Administrative Templates\Windows Components\Store\Turn off Automatic Download and Install of updates

Garanta que 'Turn off the offer to update to the latest version of Windows' esteja atribuido como 'Enabled'

Ativa ou desativa a oferta da Microsoft Store para atualizar para a versão mais recente do Windows.

O estado recomendado para esta configuração é: Ativado.

Atualizações não planejadas do sistema operacional podem levar a mais chamadas de suporte evitáveis. O departamento de TI deve gerenciar e aprovar todas as atualizações.

Garanta que 'Turn off the Store application' esteja atribuido como 'Enabled'

Essa configuração nega ou permite o acesso ao aplicativo Loja.

O estado recomendado para essa configuração é: Ativado.

Somente aplicativos aprovados pelo departamento de TI devem ser instalados. Permitir que os usuários instalem aplicativos de terceiros pode levar à perda de atualizações e a potenciais vulnerabilidades de dia zero.
